# AI emissions

Analysis to estimate the environmental impact of CoPilot use

## App

A Shiny App to estimate the environmental impact of the organisation's use of coPilot based on environmental factor estimates sourced  from https://app.powerbi.com/view?r=eyJrIjoiZjVmOTI0MmMtY2U2Mi00ZTE2LTk2MGYtY2ZjNDMzODZkMjlmIiwidCI6IjQyNmQyYThkLTljY2QtNDI1NS04OTNkLTA2ODZhMzJjMTY4ZCIsImMiOjF9

The data is filtered for GPT-5, the model which CoPilot uses. 

The data differs for simple, medium and complex messages, and for short, medium and long queries. The proportions of these are specified by the user

Data is an average of the data for the last 10 days

## Analysis
Produces a report to compare environmental impact in four scenarios
